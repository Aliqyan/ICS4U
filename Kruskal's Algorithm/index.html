Kruskal's Algorithm

Introduction

This Algorithm was created by Joseph Kruskal. Kruskal's Algorithm's finds the minimum spanning tree of a graph, meaning that it finds the smallest total edge weight to conect all vertices of the graph. Kruskal'a

Kruskal's Algorithm's finds the minimum spanning tree of a graph, meaning that it finds the smallest total edge weight to conect all vertices of the graph. Kruskal'a

Pseudo Code

				KRUSKAL(G):
				1 A = ∅
				2 foreach v ∈ G.V:
				3    MAKE-SET(v)
				4 foreach (u, v) in G.E ordered by weight(u, v), increasing:
				5    if FIND-SET(u) ≠ FIND-SET(v):
				6       A = A ∪ {(u, v)}
				7       UNION(u, v)
				8 return A
